# @kb-labs/core

**Status:** MVP 1.0  
**Category:** Core Platform  
**Repository:** [kb-labs-core](https://github.com/KirillBaranov/kb-labs-core)

## Overview

Runtime core with profiles resolver/validator and infrastructure abstractions. Provides essential utilities, system interfaces, and shared functionality across the KB Labs ecosystem.

## Key Features

- **Profile System** - Resolver/validator with 6-layer configuration merge
- **Bundle API** - Single entry point for loading complete product bundles
- **Infrastructure Abstractions** - Logging, filesystem, repository interfaces
- **Core Packages** - config, profiles, bundle, CLI, sys, types, policy
