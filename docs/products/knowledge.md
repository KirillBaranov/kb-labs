# @kb-labs/knowledge

**Status:** MVP 1.0
**Category:** Core Platform
**Repository:** [kb-labs-knowledge](https://github.com/KirillBaranov/kb-labs-knowledge)

## Overview

Knowledge management contracts and runtime for KB Labs projects. Provides standardized knowledge artifacts, reusable knowledge presets, and deep integration with AI products across the ecosystem.

## Key Features

- **Knowledge Contracts** - Standardized interfaces for knowledge artifacts
- **Artifact Management** - Rules, handbooks, ADRs, and boundaries storage
- **Preset System** - Reusable knowledge configurations
- **AI Integration** - Deep integration with AI Review, AI Docs, AI Tests
