# @kb-labs/product-template

**Status:** MVP 1.0  
**Category:** Templates & Tools  
**Repository:** [kb-labs-product-template](https://github.com/KirillBaranov/kb-labs-product-template)

## Overview

Baseline template for products under the @kb-labs namespace. Fast bootstrap, unified quality rules, simple publishing, and reusable core utilities.

## Key Features

- **Fast Bootstrap** - 5-minute deployment with shared setup and architecture
- **Unified Standards** - Pre-configured with DevKit standards
- **Template Structure** - Multi-package repository structure with pnpm workspaces
- **Documentation** - Includes ADR templates and documentation standards
- **CI/CD Ready** - Pre-configured GitHub Actions workflows
