# @kb-labs/studio

**Status:** MVP 1.0  
**Category:** Tools & Infrastructure  
**Repository:** [kb-labs-studio](https://github.com/KirillBaranov/kb-labs-studio)

## Overview

Unified web dashboard for observing and managing the KB Labs ecosystem. React-based web application that provides observability and management capabilities.

## Key Features

- **Web Dashboard** - Unified interface for all KB Labs tools
- **Observability** - Real-time monitoring and visualization
- **Management** - Configuration and control capabilities
- **React-Based** - Modern React application with Ant Design components
