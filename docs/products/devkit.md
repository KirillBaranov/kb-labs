# @kb-labs/devkit

**Status:** MVP 1.0  
**Category:** Core Platform  
**Repository:** [kb-labs-devkit](https://github.com/KirillBaranov/kb-labs-devkit)

## Overview

Bootstrap and standards for the KB Labs ecosystem. Provides TypeScript configs, ESLint, Prettier, Vitest, Tsup presets, and reusable GitHub Actions workflows.

## Key Features

- **TypeScript Presets** - Ready-to-use tsconfig for libraries, Node services, and CLIs
- **ESLint Configurations** - Base preset plus variations for different project types
- **Prettier** - Opinionated formatting profile
- **Vitest** - Test configuration presets
- **GitHub Actions** - Reusable CI/CD workflows
- **Sync System** - Automatic configuration synchronization
