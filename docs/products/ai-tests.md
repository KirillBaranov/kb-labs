# @kb-labs/ai-tests

**Status:** MVP 1.0
**Category:** AI Products
**Repository:** [kb-labs-ai-tests](https://github.com/KirillBaranov/kb-labs-ai-tests)

## Overview

Plan, generate, run, repair, and audit automated tests powered by KB Mind context. AI-powered test generation and maintenance for multiple testing frameworks.

## Key Features

- **Test Planning** - Plan test suites based on code structure and coverage gaps
- **Test Generation** - AI-powered test generation from code and specifications using Mind context
- **Test Running** - Execute tests and collect results
- **Test Repair** - Automatically repair broken tests when code changes
- **Coverage Audit** - Analyze and optimize test coverage
- **Multi-Framework** - Support for multiple testing frameworks (Vitest, Jest, Playwright)
- **Mind Integration** - Uses Mind RAG for intelligent, context-aware test generation

## Commands

| Command | Purpose |
|---------|---------|
| `kb ai-tests:plan` | Plan test suites based on code |
| `kb ai-tests:generate` | Generate tests from code |
| `kb ai-tests:run` | Execute tests |
| `kb ai-tests:repair` | Repair broken tests |
| `kb ai-tests:audit` | Audit test coverage |
