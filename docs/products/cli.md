# @kb-labs/cli

**Status:** MVP 1.0  
**Category:** Core Platform  
**Repository:** [kb-labs-cli](https://github.com/KirillBaranov/kb-labs-cli)

## Overview

UX wrapper over core providing unified CLI commands (kb *). Enables fast bootstrap, unified quality rules, simple publishing, and reusable core across all KB Labs projects.

## Key Features

- **Unified CLI** - Single command interface for all KB Labs tools
- **Command System** - Extensible command registration and execution
- **Output Formatting** - Consistent, formatted output across commands
- **Adapters** - Integration adapters for external systems
