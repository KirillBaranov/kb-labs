# @kb-labs/ui

**Status:** MVP 1.0  
**Category:** Tools & Infrastructure  
**Repository:** [kb-labs-ui](https://github.com/KirillBaranov/kb-labs-ui)

## Overview

Shared UI component library for KB Labs ecosystem. Framework-agnostic design system foundation with React component implementations.

## Key Features

- **Design System** - Consistent design tokens and components
- **React Components** - Reusable React component library
- **Framework-Agnostic** - Design tokens work with any framework
- **Accessibility** - Built with accessibility in mind
