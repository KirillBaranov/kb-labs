# @kb-labs/analytics

**Status:** MVP 1.0  
**Category:** AI Products  
**Repository:** [kb-labs-analytics](https://github.com/KirillBaranov/kb-labs-analytics)

## Overview

Unified event analytics pipeline for the KB Labs ecosystem. Low-overhead, extensible event pipeline for collecting, processing, and shipping structured events.

## Key Features

- **Event Pipeline** - Low-overhead event collection and processing
- **Pluggable Sinks** - Multiple output destinations (console, file, API, database)
- **Backpressure Control** - Handles high-volume event streams
- **Singleton SDK** - Single instance pattern for efficient resource usage
- **Type-Safe Events** - Strongly typed event definitions
