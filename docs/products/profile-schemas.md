# @kb-labs/profile-schemas

**Status:** MVP 1.0  
**Category:** Core Platform  
**Repository:** [kb-labs-profile-schemas](https://github.com/KirillBaranov/kb-labs-profile-schemas)

## Overview

JSON Schema contracts and profiles for KB Labs ecosystem. Provides validation, consistency, and compatibility between products (AI Review, AI Docs, AI Tests, AI Assistant, etc.).

## Key Features

- **JSON Schemas** - Validation schemas for profiles, rules, and products
- **Type Definitions** - TypeScript types generated from schemas
- **Compatibility** - Ensures consistency across all KB Labs products
- **Validation** - Runtime validation of profile structures
