# @kb-labs/mind

**Status:** MVP 1.0  
**Category:** AI Products  
**Repository:** [kb-labs-mind](https://github.com/KirillBaranov/kb-labs-mind)

## Overview

Headless context layer for KB Labs projects. Provides intelligent code indexing, dependency tracking, and context pack generation for AI-powered development workflows.

## Key Features

- **Code Indexing** - Intelligent indexing of codebase structure and dependencies
- **Context Generation** - Creates structured context packs for AI tools
- **Dependency Tracking** - Maps dependencies between modules and packages
- **LLM-Optimized** - Optimized for consumption by AI tools like Cursor
