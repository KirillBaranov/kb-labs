# @kb-labs/shared

**Status:** MVP 1.0  
**Category:** Core Platform  
**Repository:** [kb-labs-shared](https://github.com/KirillBaranov/kb-labs-shared)

## Overview

Common types and utilities without side effects for all KB Labs products. Serves as the shared foundation for product-agnostic domain utilities.

## Key Features

- **Boundaries** - Module import boundary checker with configurable rules
- **Diff Parser** - Unified diff parser and analysis utilities
- **Profiles** - Project profile management and resolution
- **TextOps** - Text processing, normalization, and similarity utilities
