# @kb-labs/plugin-template

**Status:** MVP 1.0
**Category:** Templates
**Repository:** [kb-labs-plugin-template](https://github.com/KirillBaranov/kb-labs-plugin-template)

## Overview

Gold standard reference template for building production-ready KB Labs plugins with CLI, REST, and Studio surfaces. Contains 7,700+ lines of comprehensive documentation and 100% canonical patterns.

## Key Features

- **7,700+ Lines of Documentation** - Comprehensive guides, examples, and best practices
- **100% Canonical Patterns** - No legacy code, all modern best practices
- **Production-Ready Examples** - Validation, testing, contracts, multi-tenancy
- **Complete Error Handling** - 8 custom error classes with proper handling
- **KB Labs Standard Structure** - cli, rest, studio, lifecycle, core, utils folders
- **Multi-Surface Support** - CLI commands, REST endpoints, Studio UI components
- **Lifecycle Hooks** - Setup, install, update, uninstall lifecycle management

## What's Included

- Complete plugin manifest V2 example
- CLI command implementations
- REST API endpoints
- Studio UI components
- Testing setup with Vitest
- Error handling patterns
- Multi-tenancy support
- Validation with Zod schemas
- TypeScript configuration
- Build configuration with tsup
